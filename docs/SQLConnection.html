Class: SQLConnection

SQLConnection(db, edm)

Соединение или контроллер базы данных. Вся работа с базой данных должна осуществляться через с помощью экземпляра класса. Этот класс, хотя и содержит реализацию для postgre, может использоваться только как прототип для контроллеров разных типов баз.
