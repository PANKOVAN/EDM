Class: DataController

DataController(name, req, res)

Контроллер данных. Используется с одной строны как поставщик данных на клиент, с другой как базовый класс для всех специальных контроллеров данных.

Контроллер данных умеет выполнять запросы к базе данных по имени класса и переданным параметрам. Ответственность за выделение имени класса и форирование параметров лежит на роудере данных из которого предполагается вызывать контроллер. Функциональность контроллера по начитке данных ограничевается возможностями автоматической генерации EDM. Если этой функциональности не хватает , то можно создать новый класс контроллера, используя в качестве прототипа DataController
